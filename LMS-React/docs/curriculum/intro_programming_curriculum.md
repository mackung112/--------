# Introductory Computer Programming Curriculum (21910-1003)

## Unit 1: หลักการเขียนโปรแกรมเบื้องต้น
- 1.1 ความหมายของโปรแกรมและภาษาคอมพิวเตอร์
- 1.2 ระดับของภาษาคอมพิวเตอร์
- 1.3 ตัวแปลภาษาแบบ Interpreter และ Compiler
- 1.4 วิวัฒนาการและจุดเด่นของภาษา Python
- 1.5 การติดตั้ง Python Interpreter และเครื่องมือพัฒนา (IDE)
- 1.6 หลักการเขียนโปรแกรมที่สะอาด (Pythonic Way)
- 1.7 โครงสร้างการจัดย่อหน้า (Indentation) และบรรทัดคำสั่ง

## Unit 2: ขั้นตอนการเขียนโปรแกรม
- 2.1 ความหมายของอัลกอริทึม (Algorithm)
- 2.2 วงจรการพัฒนาระบบ (SDLC)
- 2.3 สัญลักษณ์มาตรฐานของผังงาน (Flowchart Symbols)
- 2.4 การออกแบบผังงานแบบเรียงลำดับ (Sequence Flowchart)
- 2.5 การออกแบบผังงานแบบมีทางเลือก (Selection Flowchart)
- 2.6 การออกแบบผังงานแบบวนซ้ำ (Iteration Flowchart)
- 2.7 คุณลักษณะและข้อจำกัดของผังงานที่ดี
- 2.8 หลักการเขียนรหัสเทียม (Pseudocode) และการใช้คำกริยาที่เป็นกลาง
- 2.9 การใช้คำหลัก START, READ, COMPUTE, PRINT ในรหัสเทียม
- 2.10 การเขียนรหัสเทียมสำหรับเงื่อนไข (IF...THEN...ELSE)
- 2.11 การเขียนรหัสเทียมสำหรับการวนซ้ำ (WHILE...DO)
- 2.12 การจัดย่อหน้าและความเป็นระเบียบของรหัสเทียม

## Unit 3: โครงสร้างพื้นฐานของภาษา Python
- 3.1 โครงสร้างของภาษา Python
- 3.2 กฎการตั้งชื่อตัวแปร (Naming Convention)
- 3.3 ข้อมูลประเภทตัวเลขจำนวนเต็ม (Integer)
- 3.4 ข้อมูลประเภทตัวเลขทศนิยม (Float)
- 3.5 ข้อมูลประเภทข้อความ (String)
- 3.6 ข้อมูลประเภทตรรกศาสตร์ (Boolean)
- 3.7 การนำเข้าโมดูลมาตรฐานด้วยคำสั่ง import
- 3.8 การจัดการหน่วยความจำ (Memory Management)

## Unit 4: การใช้กระบวนการเขียนโปรแกรม
- 4.1 ภาพรวมการควบคุมทิศทางโปรแกรม
- 4.2 การรับข้อมูลด้วยคำสั่ง input()
- 4.3 การแสดงผลและการจัดรูปแบบข้อความด้วยคำสั่ง print()
- 4.4 ตัวดำเนินการทางคณิตศาสตร์ (Arithmetic Operators)
- 4.5 ตัวดำเนินการกำหนดค่า (Assignment Operators)
- 4.6 ตัวดำเนินการเปรียบเทียบ (Comparison Operators)
- 4.7 ตัวดำเนินการทางตรรกศาสตร์ (Logical Operators)
- 4.8 ตัวดำเนินการระบุเอกลักษณ์ (Identity Operators)
- 4.9 ตัวดำเนินการความเป็นสมาชิก (Membership Operators)
- 4.10 ตัวดำเนินการระดับบิต (Bitwise Operators)
- 4.11 การใช้งานเงื่อนไข if
- 4.12 การใช้งานเงื่อนไข elif
- 4.13 การใช้งานเงื่อนไข else
- 4.14 การทำงานซ้ำด้วย while loop
- 4.15 การทำงานซ้ำด้วย for loop
- 4.16 การใช้งานฟังก์ชัน range() ร่วมกับลูป

## Unit 5: การจัดการข้อมูลแบบชุด
- 5.1 การใช้งาน List (การเพิ่ม ลบ และแก้ไขข้อมูล)
- 5.2 การใช้งาน Tuple และ Dictionary (Key-Value Pairs)
- 5.3 การใช้งาน Set และการจัดการข้อความ (String Manipulation/Slicing)

## Unit 6: ฟังก์ชัน (Function)
- 6.1 การเรียกใช้งานฟังก์ชันมาตรฐาน (Built-in Functions)
- 6.2 การสร้างฟังก์ชันใช้งานเองด้วยคำสั่ง def และการส่งค่าพารามิเตอร์
- 6.3 การคืนค่าจากฟังก์ชัน (return) และขอบเขตตัวแปร (Local vs Global)
- 6.4 ทดลองสร้างฟังก์ชันและทดสอบผลลัพธ์

## Unit 7: การออกแบบโปรแกรมอย่างง่าย
- 7.1 หลักการออกแบบโปรแกรมทางธุรกิจ
- 7.2 การเขียนโปรแกรมธุรกิจ (มีเงื่อนไขและวนซ้ำ)
- 7.3 การจัดการข้อผิดพลาด (Error Handling)
- 7.4 การทำงานกับไฟล์ (File Handling)

## Unit 8: การประยุกต์ใช้งาน
- 8.1 การจัดทำเอกสารประกอบโปรแกรม
- 8.2 โครงงานเขียนโปรแกรมอย่างง่าย (ระบบ Mini POS)

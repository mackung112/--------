# OOP Course Curriculum (21910-1013)

## Unit 1: การเตรียมเครื่องมือ
- 1.1 ติดตั้ง Python
- 1.2 ติดตั้ง VS Code
- 1.3 ติดตั้ง Extension Python
- 1.4 ตั้งค่า Python Interpreter
- 1.5 ใช้งาน Terminal ใน VS Code
- 1.6 ติดตั้งไลบรารีด้วย pip
- 1.7 จัดการโฟลเดอร์โปรเจกต์

## Unit 2: คลาสและวัตถุ (Classes & Objects)
- 2.1 การเขียนคลาส (class)
- 2.2 กำหนดคุณสมบัติ (__init__)
- 2.3 การใช้งาน self
- 2.4 การสร้างเมธอด (Methods)
- 2.5 การสร้างออบเจ็กต์ (Instantiation)
- 2.6 การซ่อนข้อมูล (_ และ __)
- 2.7 การใช้งาน @property
- 2.8 การเก็บวัตถุใน List
- 2.9 การสืบทอดคลาส (Inheritance)

## Unit 3: การออกแบบ GUI
- 3.1 พื้นฐานไลบรารี Tkinter
- 3.2 การสร้างหน้าต่างหลัก
- 3.3 ตั้งค่าขนาดและชื่อหน้าต่าง
- 3.4 การสร้างป้ายชื่อ (Label)
- 3.5 การสร้างช่องกรอก (Entry)
- 3.6 การสร้างปุ่มกด (Button)
- 3.7 การจัดวางแบบ Grid และ Pack
- 3.8 การสร้างกล่องเลือก (Combo)
- 3.9 การปรับแต่งสีและฟอนต์

## Unit 4: การทดสอบโค้ด
- 4.1 ดักจับ Error (Try-Except)
- 4.2 ใช้ Debugger ใน VS Code
- 4.3 การอ่าน Error ใน Terminal
- 4.4 การเขียน Unit Test
- 4.5 การแปลงชนิดข้อมูล (Casting)
- 4.6 การตรวจสอบข้อมูล (Validation)
- 4.7 การบันทึกไฟล์ JSON
- 4.8 การทดสอบตรรกะคำนวณ

- 5.1 วิเคราะห์โจทย์ธุรกิจ
- 5.2 เขียนแผนผังคลาส (UML)
- 5.3 การแยกไฟล์โมเดลข้อมูล
- 5.4 การเชื่อม GUI กับตรรกะ
- 5.5 พัฒนาระบบขายของ (POS)
- 5.6 การทดสอบระบบรวม (E2E)

- 5.8 การนำเสนอและสาธิตผลงาน

# 🎯 แผนงานและสถานะปัจจุบัน (Task Management - ค่าเริ่มต้น)

**เป้าหมาย:** สร้างเนื้อหาบทเรียนวิชาระบบปฏิบัติการ บำรุงรักษา และเครือข่ายคอมพิวเตอร์เบื้องต้น (และรายวิชาอื่นๆ) ให้สมบูรณ์แบบโดยเน้นสื่อการเรียนรู้เชิงโต้ตอบระดับพรีเมียม (Interactive Simulator) และปฏิบัติตามกฎเกณฑ์ใน "ค่าเริ่มต้น" (AGENTS.md, CONTEXT.md, DESIGN.md, SKILL.md, task.md) อย่างเคร่งครัด


---

## 📌 1. เป้าหมายปัจจุบัน (Active Objectives)
- [ ] พัฒนาบทเรียนหน่วยการเรียนรู้ถัดไป **"หน่วยที่ 2 ฮาร์ดแวร์และการประกอบคอมพิวเตอร์"** (ครอบคลุมหัวข้อ 2.1 ถึง 2.4) ในระบบ LMS-React

## 🔄 2. รายการงานที่กำลังดำเนินการและแผนงานถัดไป (Active Checklist)
- [ ] ค้นคว้ารายละเอียดสเปกหลักสูตรวิชาการเชิงลึกของหน่วยที่ 2.1 (CPU สถาปัตยกรรม, เมนบอร์ด, แรม, สตอเรจ, พาวเวอร์ซัพพลาย)
- [ ] วางแผนพัฒนาเครื่องจำลองการถอดประกอบคอมพิวเตอร์เชิงตอบสนอง (Computer Assembly Interactive Simulator)
- [ ] เตรียมโจทย์ปฏิบัติงานของเครื่องมือและความปลอดภัยด้านไฟฟ้าและไฟฟ้าสถิต (ESD Prevention)

### 🔄 พัฒนาบทเรียนย่อย 1.2, 1.3, 1.4 (ทิศทางส่งสัญญาณ, สถาปัตยกรรมสิทธิ์, และ Physical Topologies) - [เสร็จสมบูรณ์ 30 May 2026]
- **สิ่งที่ทำ**: ดำเนินการพัฒนาระบบอัตโนมัติเนื้อหาและเครื่องจำลองอัจฉริยะเชิงตอบสนอง 3 หน่วยการเรียนรู้หลักต่อเนื่อง ร่างโค้ดใหม่ 100% จากศูนย์ลุล่วงอย่างไร้ placeholders บิลด์ผ่านผ่านด้วยเวลา 1.01 วินาที!
  - **Unit 1.2 (ความรู้เบื้องต้นเครือข่าย)**: พัฒนาเครื่องจำลอง 3 โหมด (*Simplex*, *Half-Duplex*, *Full-Duplex*) พร้อมการจำลองบิตกระแสสัญญาณไฟฟ้าชนกันระเบิดกึ่งกลางสายส่ง (x = 300) และรายงานตรรกะ CSMA/CD ลงใน Console
  - **Unit 1.3 (ประเภทและสถาปัตยกรรม)**: พัฒนาเครื่องจำลอง 3 โหมด (*Client-Server* จำลอง Switch ล่ม/SPOF, *Peer-to-Peer* ติ๊กสลับออนไลน์เพื่ออ้อมสัญญาณสำรอง, *Security Venn Zones* จำลองเกตเวย์ดักกรอง 3 ระดับความลับข้อมูล)
  - **Unit 1.4 (รูปแบบการเชื่อมต่อ - Topology)**: พัฒนาเครื่องจำลอง 4 โหมด (*Bus* จำลองกระแสชน Jamming, *Star* วิเคราะห์จุดล้มเหลวเดี่ยว Switch, *Ring* ระบบเหรียญส่ง Token Passing หมุน 360 องศาเพื่อความเสถียร, *Mesh* ติ๊กตัดสายสัญญาณหลัก A-C และสับเปลี่ยนเส้นทางอ้อมระบบ Mesh)
  - **มาตรฐานสุนทรียภาพ**: คอนเนคเตอร์และเส้นทาง SVG วิ่งเข้าและออกจากศูนย์กลางทางเรขาคณิต (Absolute Center) 100% ที่พิกัดกึ่งกลาง, ปล่อยข้อความพาดลงพื้นหลัง `#FAFAFA` สะอาดโปร่งแบบไร้กรอบ (Fluid Open-Air Layout), และขนาดตัวอักษรเป็นไปตามสุขภาวะ Legible Font Scale
  - **ใบงานและการประเมินผล**: ผนวกด่านตอบคำถามระดับวิชาชีพ QuizEngine หน่วยละ 5 ด่าน พร้อม Console รายงานภาษาไทยวิชาการ และ Consolidated TeacherTask โจทย์ปฏิบัติงานการดีไซน์อุโมงค์ VPN และตรรกะระบบเกตเวย์

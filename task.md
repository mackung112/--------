    - **ส่วนที่ 2: โครงสร้างแบบทางเลือกเดี่ยว (Single If tracer)** - ตัวจำลองยอดสั่งซื้อหักส่วนลด สไลเดอร์จำลองปรับยอด (500 - 2000 บาท) เพื่อแยกไฟวิ่งสายตรรกะฝั่ง True เลี้ยวขวาและพิมพ์ผลลัพธ์ผ่าน pointed Display shape ขนาดปลอดภัย x=185
    - **ส่วนที่ 3: โครงสร้างแบบสองทางเลือก (If-Else score tracer)** - ตัวจำลองประเมินคะแนนสอบ ผ่าน/ตก สไลเดอร์ (0 - 100) สัญญาณกระพริบแยกสาย และตัวปิด Display สรุปผล
    - **ส่วนที่ 4: โครงสร้างแบบหลายทางเลือก (If-Elif-Else cascade)** - บอร์ดคัดกรองเกรดสากล (A, B, C, F) โดยจำลองกระแสไฟในผังงาน SVG วิ่งไหลเยื้องลงเป็นขั้นบันได (Cascading Diamond)
    - **ส่วนที่ 5: โครงสร้างทางเลือกซ้อนกัน (Nested If voter tracer)** - บอร์ดตรวจสอบสิทธิ์เลือกตั้ง 2 ชั้น (อายุ 15-25 ปี และ toggle สัญชาติไทย) ไฮไลท์เส้นทางตรรกะสีสันสดใส
    - **ส่วนที่ 6: TeacherTask** - กล่องกิจกรรมภาคปฏิบัติมอบหมายให้เขียนโครงสร้างหลายทางเลือกคิดคะแนนเกรดลูกค้าพรีเมียม
